# ID3--Midwest_Prediction

**The Goal ("midwest" column):** The quiz is designed to guess whether or not a person lives in the American Midwest based on their survey answers.

**The Questions:** It asks if you like various specific foods.

**The Answers (0 and 1):** Instead of typing out "yes" or "no," the quiz tracks answers as numbers. A 1 means you like the food, and a 0 means you do not.

**Algorithm Used:** ID3 algorithm

**Feature used for prediction:** apple_pie, potato_salad, sushi
